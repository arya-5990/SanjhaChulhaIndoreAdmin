# Sanjha Chulha Admin App

Admin application for managing the Sanjha Chulha restaurant website.
Built with React Native (Expo).

## Features
- **Dashboard**: Overview of restaurant stats.
- **Menu Management**: Add, edit, and delete culinary offerings.
- **Reservations**: View and manage table bookings (Approve/Reject).
- **Settings**: Configure app settings.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Configure Environment**
   Create a `config/firebase.js` with your actual Firebase credentials.
   (See `src/config/firebase.js` for the structure)

3. **Run the App**
   ```bash
   npx expo start
   ```

## Tech Stack
- **Framework**: React Native (Expo)
- **Navigation**: React Navigation (Native Stack)
- **Styling**: StyleSheet + Custom Theme (Inter Font)
- **Icons**: Lucide React Native
- **Backend**: Firebase (Auth, Firestore, Storage)

## Note
This app uses a minimalist UI design with a warm color palette tailored to the restaurant's branding.
